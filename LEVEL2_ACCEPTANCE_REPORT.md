## 🚨 已知问题

### 1. GitHub Pages 访问延迟
- **现象**: CDN缓存导致更新延迟
- **状态**: 已提供离线版，用户可本地运行
- **解决**: 添加 `.nojekyll` 文件，优化缓存策略

### 2. 代理通信系统故障
- **现象**: `sessions_spawn: agentId is not allowed`
- **根因**: subagent_spawning hooks 未注册
- **状态**: 不影响交付，用户可直接使用
- **建议**: 后续修复以启用自动化工作流
