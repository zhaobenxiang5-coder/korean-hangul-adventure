# 用户测试计划

## 测试目标
验证 Level 2 的可用性、趣味性和学习效果

## 测试对象
- 年龄：12-45岁
- 韩语基础：零基础或少量基础
- 设备：桌面/移动浏览器

## 测试流程

### 阶段1: 首次体验 (5分钟)
1. 打开 level2-offline-v2.html
2. 浏览主菜单，理解选项
3. 点击"Level 1: 元音基础"体验3个字母
4. 使用TTS发音功能

### 阶段2: 深度体验 (10分钟)
1. 完成Level 1全部字母学习
2. 尝试"拼图游戏"3次
3. 进入Level 2，体验"音节拼图"
4. 完成"单词挑战"5题

### 阶段3: 反馈收集
- 填写问卷（5分制评分）
- 记录操作难点
- 提出改进建议

## 测试指标
- 任务完成率
- 平均会话时长
- 错误操作次数
- 满意度评分

## 招募渠道
- 社交媒体（韩语学习群）
- 朋友推荐
- 在线学习社区

---

**预计样本量**: 20-30人  
**测试周期**: 3天  
**交付**: 测试报告 + 优化建议 📋
