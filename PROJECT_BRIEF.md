## 🎯 MVP需求规格

### 核心功能
1. **学习内容**: 5个基础元音 (ㅏ, ㅓ, ㅗ, ㅜ, ㅡ)
2. **游戏类型**: 字母拼图 - 拖拽字母到正确发音位置
3. **交互方式**: 鼠标/触屏拖拽 + TTS发音
4. **进度追踪**: localStorage记录完成状态
5. **视觉风格**: 简洁可爱，韩式设计元素

### 技术实现
- **前端**: Phaser 3 + HTML5 + Tailwind CSS (单文件)
- **音频**: sag TTS技能 (韩语发音)
- **部署**: GitHub Pages
- **代码托管**: GitHub + PR workflow
