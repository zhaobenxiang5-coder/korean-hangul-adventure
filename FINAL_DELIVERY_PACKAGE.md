## 🔧 已知问题与修复建议

### P0 (紧急 - 影响可用性)

| 问题 | 影响 | 建议修复 |
|------|------|----------|
| Web Speech API 在Safari需要用户交互 | iOS用户TTS可能无声音 | 添加降级UI提示 |
| 缺少错误边界 | 运行时错误导致整个App崩溃 | 包裹所有场景create/update于try-catch |
| 全局变量污染 | 可能与其他脚本冲突 | IIFE封装或ES6 Module |

### P1 (重要 - 影响体验)

| 问题 | 影响 | 建议修复 |
|------|------|----------|
| 移动端按钮尺寸偏小 | 误触率较高 | 增大到120x60px |
| 拖拽时无视觉反馈 | 操作不明确 | 添加scale+shadow效果 |
| 缺少键盘导航 | 无障碍访问受限 | 实现键盘快捷键 |
| 无ARIA标签 | 屏幕阅读器无法使用 | 添加aria-label |

### P2 (优化 - 锦上添花)

| 问题 | 影响 | 建议修复 |
|------|------|----------|
| 场景切换生硬 | 流畅度不足 | 添加fade in/out动画 |
| 颜色对比度边缘 | 部分文本可读性 | 使用#4A5568代替#666 |
| 代码无注释 | 维护成本高 | 添加JSDoc |
| 缺少API文档 | 二次开发困难 | 编写开发者文档 |
